# 🚀 Mac Optimizer Skill

## Purpose
Automates macOS performance optimization, memory management, and system cleanup. Prevents Mac from hanging and ensures maximum efficiency for AI agents and user workflows.

## Capabilities
- **CPU Optimization**: Automatically detects and kills zombie processes or CPU-throttling tasks.
- **Memory Management**: Frees inactive RAM and monitors memory pressure.
- **Disk Cleanup**: Bereinigt temporäre Dateien, Xcode-Caches und alte Logs.
- **Network Reset**: Behebt DNS-Probleme und setzt Netzwerk-Interfaces zurück.
- **Automation**: Runs as a daemon to provide continuous background optimization.

## Integration
- **Hetzner Sync**: Optimized to work alongside the Hetzner Empire server to balance load between local and remote environments.
- **Monetization**: Integrated with Gumroad/Twitter sales funnel for commercial distribution.

## Commands
- `optimize-mac`: Run full performance fix.
- `check-mac-status`: View current CPU, RAM, and Disk telemetry.
- `clean-temp`: Remove unnecessary large files.
