# X/Twitter Content Engine Skill

Automatische Content-Generierung, Scheduling und Growth Hacking fuer X/Twitter.
Agent: Kelly (Content Creator)

## Browser Script
`openclaw/browser/x-content-engine.sh`

## Befehle
- `/kelly-generate` — AI Tweet generieren (mit Post-Option)
- `/kelly-content-thread` — Thread generieren (8-12 Tweets)
- `/kelly-viral-hooks` — 20 virale Hook-Ideen
- `/kelly-schedule` — Tweets vorplanen (Queue)
- `/kelly-auto-engage` — Auto-Engagement (Smart Replies)
- `/kelly-growth` — Growth Opportunities scannen
- `/kelly-daily` — Taegliche Content-Routine (5 Schritte)
- `./x-content-engine.sh hashtags <topic>` — Trending Hashtags
- `./x-content-engine.sh repurpose <url>` — Content Repurposing
- `./x-content-engine.sh analytics` — Performance Stats

## Features
- AI-generierte Tweets mit viralen Hook-Templates
- Thread-Generator (8-12 Posts mit Hook + CTA)
- Content Queue mit Scheduling
- Auto-Engagement mit smarten Reply-Vorschlaegen
- Growth Scanner basierend auf Trending Topics
- Content Repurposing (URL → Tweet + Thread + Poll)
- Taegliche 5-Schritt Content-Routine
