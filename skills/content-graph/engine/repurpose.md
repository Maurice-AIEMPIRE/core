# engine/repurpose.md – 1 Idee → 10 Native Outputs

Repurposing bedeutet NICHT kopieren. Jede Platform denkt das Thema anders.

## Workflow

### Input
Eine Idee, Erkenntnis, Trading-Ergebnis, oder Story.

### Step 1: X Thread (Primary Asset)
Schreibe zuerst den X-Thread nach [[platforms/x.md]].
→ Das ist dein "Rohstoff" für alle anderen Outputs.

### Step 2: LinkedIn (Expand)
Nimm den Kern des X-Threads und schreibe eine Personal Narrative.
→ Nicht der Thread ausgeschrieben – neue Struktur, neues Framing.
→ Siehe [[platforms/linkedin.md]]

### Step 3: IG Carousel (Visualize)
Extrahiere die 5–7 Kernpunkte als Slides.
→ Jeder Slide = 1 Aussage, visuell denkend.
→ Siehe [[platforms/ig.md]]

### Step 4: TikTok Script (Compress)
Nimm den stärksten Insight und mache ein 45s-Script.
→ Hook-Setup-3Punkte-CTA.
→ Siehe [[platforms/tiktok.md]]

### Step 5: YouTube Outline (Deepen)
Expand den LinkedIn-Artikel zu einem 8-min-Video.
→ Mehr Kontext, Beispiele, Daten.
→ Siehe [[platforms/youtube.md]]

### Step 6: Threads (Fragment)
Nimm 3 starke Einzelaussagen aus dem X-Thread.
→ Jede als eigenständiger Threads-Post.
→ Siehe [[platforms/threads.md]]

### Step 7: Newsletter (Master Version)
Kombiniere LinkedIn + YouTube-Outline zu Deep-Dive.
→ Vollständige Ressourcen + persönliche Insights.
→ Siehe [[platforms/newsletter.md]]

### Steps 8–10 (Optional Expansion)
- Pinterest: IG-Slides umformatieren
- Podcast-Notizen: YouTube-Outline als Show-Notes
- Reddit/HN: sachlicher Ton, keine Selbst-Promotion
