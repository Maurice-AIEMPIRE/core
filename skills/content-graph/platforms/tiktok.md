# platforms/tiktok.md – TikTok Content Rules

hook muss in ersten 3 Sekunden sitzen.
ton aus [[voice/platform-tone.md]] – informell, direkt, energetisch.
audience: [[audience/casual.md]] primär.
see [[engine/hooks.md]] für Einstieg.

## Format: 45s Script
- 0–3s: Hook (visuell + verbal)
- 3–15s: Problem / Setup
- 15–35s: Solution / Core Content (3 Punkte max.)
- 35–45s: CTA ("Follow", "Save", "Kommentar")

## Script-Template
```
[HOOK - sprich direkt in Kamera]
"Ich mache X und spare damit Y – hier ist wie:"

[SETUP]
"Die meisten machen das falsch weil..."

[3 PUNKTE]
"1. [Punkt] 2. [Punkt] 3. [Punkt]"

[CTA]
"Wenn du das auch willst, folge mir für täglich mehr."
```

## Screen-Recording-Anweisung (wenn relevant)
- Tool/App öffnen, Screen zeigen
- Schritt-für-Schritt, keine langen Erklärungen
- Text-Overlays für Key-Points

## Posting-Frequenz
1x/Tag (Morgens 7–9 Uhr) – siehe [[engine/scheduling.md]]
