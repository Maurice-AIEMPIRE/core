# SKILL_content_skill_graph.md

Name: Content Skill Graph Engine (10 Platforms – Native Output)
Description: Verwandelt eine Idee in 8–10 plattform-native Posts via interconnected Markdown Graph (inspired by @DeRonin_ system).

Requirements: OpenClaw Memory + SOUL.md + AGENTS.md + Phoenix + Humanizer 24-Point + SEO Auditor

## Workflow (bei jeder Idee)

1. **Memory-Search** + `skills/content-graph/index.md` lesen (Briefing + Node-Map)
2. **Graph traversieren**:
   `[[platforms]]` → `[[voice]]` → `[[engine/hooks.md]]` → `[[engine/repurpose.md]]` → `[[audience]]`
3. **8–10 echte Varianten erstellen** (nicht Kopien – jedes Format denkt anders):
   - X: Contrarian Thread, lowercase, 5–8 Posts
   - LinkedIn: Personal Narrative, 800–1500 Wörter
   - IG: 7-Slide Carousel + Caption
   - TikTok: 45s Script + Screen-Recording-Anweisung
   - YouTube: SEO-Titel + 8-Min-Outline
   - Threads: 3 eigenständige Posts
   - Newsletter: Deep-Dive 800–1200 Wörter
4. **Humanizer 24-Point** + SEO Auditor auf alle Outputs anwenden
5. **Hybrid Revenue Loop**: Trading-Gewinne aus Hunter/Phoenix → als Content-Idee einspeisen → Auto-Posts
6. **Publish-Vorschlag** generieren (Platform + Zeit aus scheduling.md) + Confirmation abwarten
7. **Reflection** → MEMORY.md Update → Graph selbst optimieren

## Aktivierung in OpenClaw
```
Aktiviere Ultimate Executive Mode + lade SKILL_content_skill_graph +
starte Content-Graph-Run mit Thema: [DEINE IDEE]
```

## Pro-Tip
Starte mit einem Trading-Insight oder AI-Tool-Finding als Thema.
Der Graph weiß automatisch, wie er es für jede Platform adaptiert.
