AI Engine

Das KI-Gehirn im Detail

Fuenf KI-Algorithmen arbeiten zusammen fuer autonome Entscheidungen.

1. Adaptive Schwellwerte

Dynamische Schwellwerte basierend auf deinem Nutzungsmuster: Mean + 1.5 x Standardabweichung der letzten Stunde. Was bei dir normal ist, loest keinen Alarm aus.

2. Vorhersage-Engine

Lineare Regression ueber 30 Datenpunkte (5 Min). Berechnet ob ein kritischer Schwellwert in 5-10 Minuten ueberschritten wird — und greift PRAEVENTIV ein.

3. Anomalie-Erkennung

Z-Score-basiert: Jeder Wert wird gegen die statistische Verteilung verglichen. Z-Score > 2.5 = statistisch hochgradig ungewoehnlich.

4. Korrelations-Analyse

CPURAMTempDiagnoseAktion
++—App OverloadApp identifizieren + beenden
+—+Thermal ThrottleCPU-Last reduzieren
—+—Memory PressureSpeicher freigeben
+++NOTFALLEmergency-Kill + Kuehlung

5. Feedback-Loop + Lernen

Nach jeder Aktion: 30s warten, Metriken messen. Erfolgreiche Aktionen werden hoeher gewichtet, wirkungslose uebersprungen. Das System wird mit der Zeit besser.

Intelligente Eskalation

Guardian reagiert proportional. Nie zu viel. Nie zu wenig.

Memory Guardian — 4 Stufen

StufeTriggerAktionRisiko
SoftRAM > 80%System-Cache + Browser-Cache leerenKeins
MediumRAM > 85%+ Erw. Caches + Memory Pressure WarnKeins
AggressivRAM > 90%+ Low-Priority Apps beendenApps neu starten
NotfallRAM > 95%+ Top 3 RAM-Verbraucher beendenDatenverlust

Thermal Guardian — 3 Stufen

StufeTriggerAktion
Light> 80 C + steigendSpotlight pausieren, Time Machine drosseln
Medium> 90 C+ Hintergrundprozesse drosseln (renice)
Heavy> 95 C+ Prozesse pausieren (SIGSTOP), Auto-Resume 30s

Process Manager — Schutz-Hierarchie

PrioAppsBehandlung
0 GesperrtWindowServer, kernel_task, Finder, DockWerden NIEMALS angefasst
1 KritischTerminal, Xcode, aktiver BrowserHoechste CPU-Prioritaet
2-3 NormalMail, Messages, Dev-ToolsStandard-Prioritaet
4-5 NiedrigSlack, Spotify, Docker, SteamWerden zuerst gedrosselt/beendet

Globale Sicherheitsregeln

  • Max. 5 automatische App-Beendigungen pro Stunde (Schutz vor Endlos-Kill)
  • Geschuetzte Apps werden NIEMALS beendet (konfigurierbar)
  • Vordergrund-App bekommt immer Prioritaet (erkannt via AppleScript)
  • Nach jeder Aktion: 30s Evaluation ob sie gewirkt hat
  • Alle Aktionen werden geloggt mit Zeitstempel und Ergebnis
